# prototype-2D-platformer-about-dog

A small 2D game prototype made in C++.

The project is focused on gameplay mechanics, player movement, interaction systems, and environmental learning without explicit tutorials.

Current features:
- Player movement
- Crawling mechanic
- Collision system
- Interactive objects
- Early level prototype
- Basic gameplay scenarios

Current development goals:
- Enemy dog behavior
- Distraction mechanics
- Chase sequences
- Improved level flow and interactions

This project is currently in active development and used as a learning and gameplay design prototype.
